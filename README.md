## 🛠️ Cara Menjalankan Code C++
Jika ingin menjalankan code C++ di repository ini:
1. Pastikan sudah menginstal compiler C++ (seperti MinGW / GCC).
2. Compile file menggunakan terminal: `g++ nama_file.cpp -o nama_file`
3. Jalankan program: `./nama_file` (Linux/Mac) atau `nama_file.exe` (Windows)
